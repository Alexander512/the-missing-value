Functions included in the library

the minimum function

description:

function that returns the minimum value in a dataset

syntax:

minimum(...values)

parameters:

...values

which can take the form of either a series of values:  value_1, value_2, ..., value_n

or

an array utilizing the spread operator:  ...array

return value:

a single value of type number
