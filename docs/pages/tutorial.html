Tutorial

Destructure the necessary functions from the library.

> const { mean } = require('the-missing-value.js');

The functions can accept multiple arguments due to the rest parameters. As an example, you can pass a number of integers to the mean function.

> mean(1, 2, 3, 4, 5); // returns 3

Additionally, with the use of the spread operator the functions can work with the values from one or more arrays.

> const array1 = [ 1, 2, 3, 4, 5 ];

> const array2 = [ 6, 7, 8, 9, 10 ];

> mean(...array1); // returns 3

and

> mean(...array1, ...array2); // returns 5.5
